# Diane Emerita — Official Website
 
> **Beta Release** — This site is currently in active development. Some features may be incomplete or subject to change before the official launch.
 
## Overview
 
Official website for Diane Emerita, Swedish-German singer-songwriter and multi-instrumentalist. Built as a single-page application featuring tour dates, music, videos, merch, and contact information.
 
## Tech Stack
 
- React + TypeScript
- Vite
- Tailwind CSS

## Deployment
 
Hosted on GitHub Pages. Deployed via the `gh-pages` branch. Domain managed through one.com.
